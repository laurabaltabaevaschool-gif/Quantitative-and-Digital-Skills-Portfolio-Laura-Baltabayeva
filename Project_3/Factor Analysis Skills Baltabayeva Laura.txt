The goal of this project is to apply Exploratory Factor Analysis to a dataset of airline passenger ratings to identify the underlying dimensions driving customer experience. By reducing 11 observed variables into 4 latent factors the analysis provides a clearer framework for business decision-making. The technical workflow included data cleaning, correlation analysis, and factor extraction using the Varimax rotation method in Excel. This process successfully grouped the service attributes into four distinct categories which are In-flight Comfort, Digital Experience, Service Quality, and Physical Space. The repository includes the main analysis file with all statistical tables and a formal synopsis report summarizing the business implications of the findings. This approach demonstrates how data reduction techniques can transform complex survey results into actionable insights.
